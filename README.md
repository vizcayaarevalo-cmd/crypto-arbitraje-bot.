# 🎯 Crypto Arbitrage Sniper Bot

Bot de arbitraje triangular automatizado diseñado para **Gate.io**, optimizado para ejecutar ráfagas de alta frecuencia entre pares de USDT y BTC.

## 🚀 Características
- **Multi-Scanner**: Monitoreo simultáneo de 22 activos volátiles.
- **Lógica Agresiva**: Ejecución de órdenes con umbrales de ganancia de micro-centavos ($0.001).
- **Auto-Backup**: Sincronización automática con GitHub cada vez que se alcanza un hito de balance.
- **Heartbeat System**: Monitoreo de salud del proceso en tiempo real.

## 🛠️ Stack Técnico
- **Lenguaje**: Python 3
- **Librería**: CCXT (CryptoCurrency eXchange Trading Library)
- **Infraestructura**: Linux / Docker-ready

## 📊 Historial
El bot mantiene un registro detallado de cada operación en `auto_trader.log`.
